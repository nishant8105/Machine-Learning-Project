Focused on implementing Linear Regression using Gradient Descent and comparing results with a built-in model.

## What I Did

### gradient_descent.py
- Implemented Linear Regression using Gradient Descent from scratch
- Initialized parameters (m and b) and updated them iteratively
- Calculated gradients for slope and intercept
- Tuned learning rate and number of iterations
- Visualized regression line against actual data
- Compared results with sklearn LinearRegression model

## Key Learnings
- How Gradient Descent optimizes parameters step-by-step
- Role of learning rate in convergence
- Difference between analytical solution and iterative approach
- Importance of multiple iterations for better accuracy
