## Key Learnings
- Logistic Regression is used for classification, not regression
- Output is a probability between 0 and 1
- Sigmoid function maps any value to a range between 0 and 1
- Decision boundary helps classify data into categories
- Works well for binary classification problems

## 📐 Sigmoid Function

z = 1 / (1 + e<sup>-x</sup>)

Where:
- x → input value  
- e → Euler’s number (~2.718)  
- z → output probability (between 0 and 1)
